1. Откройте терминал в папке проекта.
2. Запустите команду pip install -r requirements.txt
3. Для создания БД запустите скрипт python make_mongo.py из удобного IDE
4. В папку known_people добавьте фото лиц, которое нужно распознавать
5. Присвойте фотографиям те имена, которые должны отображаться при распознавании 
6. Для добавления имён и векторных представлений фото в БД запустите python enroll.py из удобного IDE
7. Для запуска программы запустите python recognition.py из удобного IDE
8. Для просмотра журнала запустите команду make_journal.py из удобного IDE
9. Для выхода из программы нажмите q
